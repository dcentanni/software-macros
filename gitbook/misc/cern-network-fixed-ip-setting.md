# CERN Network fixed IP setting

This page shows how to set a fixed IP address of a device connected to the CERN network (via ETHERNET).

First thing is to head to the [LanDB Portal](https://landb.cern.ch/portal) and search for device you want to set or create a new device. You can find a new device by using either its name or the MAC address if it has been already registered.

Once you found the device click on the button "New fixed IP address", you will be redirected to a new page: please set the location, outlet and all other necessary settings then click on "Send Request: NEW CONNECTION >>>".

Now a new fixed IP address will be set to the selected device, last thing to do is to couple the IP address to the ETHERNET/WIRELESS card of the selected device. This can be done by clicking on "Edit" in the "Network Interface Cards" section and select the devicename.cern.ch from the list in "Bound to IP address" then save all.
